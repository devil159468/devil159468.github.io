# NodeJS

## Koa

### Koa安装
```bash
yarn add koa // koa
yarn add nodemon // 自动重启
yarn add koa-json // json pretty
```

### 添加路由
```bash
yarn add koa-router // koa 路由
yarn add koa-ejs // koa 模板引擎
```

### POST解析
```bash
yarn add koa-bodyparser
```

### 数据库相关
```bash
yarn add mongoose
```

### 隐藏密码
```bash
yarn add bcryptjs
```

### 默认头像
```bash
yarn add gravatar
```

### JWT
```bash
yarn add jsonwebtoken
```

### 验证JWT
```bash
yarn add koa-passport
yarn add passport-jwt
```

### Pm2全局安装
```bash
yarn global add pm2
```
